 ### Setup ###

* Install Dependencies: `apt-get install npm nodejs-legacy git`
* Clone Repository: `git clone https://github.com/fanout96/recamsserver.git`
* Install Nodejs Dependencies: `npm install`

* To run: `node main.js`
